# DB Designer - Візуальний конструктор баз даних

Онлайн-інструмент для візуального створення та управління схемами баз даних.

## Функції

- 📊 Excel-подібний інтерфейс
- 📑 Вкладки для кожної таблиці
- 💾 Збереження в localStorage
- 📥 Імпорт JSON файлів
- 📤 Експорт в SQLite, MySQL, PostgreSQL, JSON
- ☑️ Чекбокси для булевих значень

## Локальний запуск

```bash
pip install -r requirements.txt
python app.py
```

Відкрийте http://127.0.0.1:5000

## Файли проекту

- `app.py` - Flask сервер
- `templates/index.html` - HTML шаблон
- `static/style.css` - Стилі
- `static/script.js` - JavaScript
- `requirements.txt` - Залежності Python
