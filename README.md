# DB Designer - Візуальний конструктор баз даних

Онлайн-інструмент для візуального створення та управління схемами баз даних.

## Функції

- 📊 Excel-подібний інтерфейс
- 📑 Вкладки для кожної таблиці
- 💾 Збереження в localStorage
- 📥 Імпорт JSON файлів
- 📤 Експорт в SQLite, MySQL, PostgreSQL, JSON
- ☑️ Чекбокси для булевих значень

## Локальний запуск

```bash
pip install -r requirements.txt
python app.py
```

Відкрийте http://127.0.0.1:5000

## Розгортання на Render (безкоштовно) ⭐ РЕКОМЕНДОВАНО

### Переваги Render:
- ✅ Повністю безкоштовний
- ✅ Автоматичний HTTPS
- ✅ Автоматичне розгортання з GitHub
- ✅ Простий інтерфейс
- ✅ Швидке розгортання (2-3 хвилини)

### Крок 1: Реєстрація
1. Зареєструйтесь на https://render.com (безкоштовно)
2. Можете увійти через GitHub акаунт

### Крок 2: Створення нового Web Service
1. Натисніть **New +** → **Web Service**
2. Виберіть **Build and deploy from a Git repository**

### Крок 3: Підключення GitHub
1. Підключіть ваш GitHub акаунт
2. Створіть новий репозиторій з файлами проекту
3. Або використайте **Public Git repository** і вкажіть URL

### Крок 4: Налаштування сервісу
Заповніть поля:
- **Name**: `db-designer` (або будь-яка назва)
- **Region**: `Ohio (US East)` або `Frankfurt (EU Central)`
- **Branch**: `main` або `master`
- **Root Directory**: залиште порожнім
- **Runtime**: `Python 3`
- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `gunicorn app:app`

### Крок 5: Запуск
1. Натисніть **Create Web Service**
2. Зачекайте 2-3 хвилини для розгортання
3. Ваш додаток буде доступний за адресою: `https://db-designer.onrender.com`

### Важливі примітки
- Перше завантаження може зайняти до 30 секунд (безкоштовний план)
- Після 15 хвилин неактивності сервіс "засинає"
- Автоматичне оновлення при push в GitHub
- Безкоштовний SSL сертифікат

### Альтернатива: Розгортання з ZIP
Якщо не хочете використовувати GitHub:
1. Завантажте `db-designer.zip`
2. Розпакуйте та створіть GitHub репозиторій
3. Дотримуйтесь інструкцій вище
